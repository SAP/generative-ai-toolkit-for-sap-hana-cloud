useful for when you need to get example to perform AutoML Classification/AutomaticClassification prediction.
------
Strictly use the Examples to create AutoML prediction instance.
Do prediction:
>>> from hana_ml.algorithms.pal.auto_ml import AutomaticClassification
>>> model_predict = AutomaticClassification()
>>> model_df = df.connection_context.table(<model_table>)
>>> model_predict.predict(data=df, key=<key>, features=<features>, model=model_df)
>>> print({"output table names": model_predict._predict_output_table_names, "Python object address": id(model_predict)})
