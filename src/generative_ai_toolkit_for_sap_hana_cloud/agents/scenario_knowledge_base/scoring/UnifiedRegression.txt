useful for when you need to get example to perform Unified Regression scoring.

Strictly use the Examples to create unified Regression scoring instance.
Do fitting:
>>> from hana_ml.algorithms.pal.unified_Regression import UnifiedRegression
>>> model_score = UnifiedRegression(func=<func>)
>>> model_df = df.connection_context.table(<model_table>)
>>> model_score.score(data=df, key=<key>, features=<features>, model=model_df, label=<label>)
>>> print({"output table names": model_score._score_output_table_names, "Python object address": id(model_score)})
